# MVP Requirements

## Must-have features
- Roadmap CRUD (create, read, update, delete roadmaps).
- Milestones with tasks per roadmap.
- Check/uncheck tasks.
- Progress percentage based on task completion.
- Per-roadmap shareable link.

## Nice-to-have features
- AI insights.
- Timeline variance analysis.
- Admin analytics.
- Reminders.

## MVP decision
The MVP will ship with the must-have features listed above. Nice-to-have features are deferred until after MVP validation.
